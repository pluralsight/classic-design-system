## Examples

### Loading

Replaces the search icon with a loading indicator

```typescript
import React from 'react'
import SearchInput from '@pluralsight/ps-design-system-searchinput'

const Comp = () => (
  <SearchInput loading placeholder="What are you looking for?" />
)

export default Comp
```

### Clear icon

Providing a `onClear` prop will display a close icon.

```typescript
import React from 'react'
import SearchInput from '@pluralsight/ps-design-system-searchinput'

const Comp = () => (
  <SearchInput onClear={() => null} placeholder="Searching for?" />
)

export default Comp
```

### Guidelines

Exercise consistency and clarity with explicit placeholder text.

<Guideline
  do={<SearchInput placeholder="Search for people" />}
  dont={<SearchInput placeholder="Filter" />}
/>

Use consistent theming to encourage visual hierarchy

<Guideline
  do={
    <div
      style={{
        background: colorsBackgroundDark[3],
        borderRadius: '4px',
        padding: layout.spacingLarge
      }}
    >
      <Theme>
        <SearchInput
          placeholder="Search for people"
          style={{ display: 'block', width: '100%' }}
        />
      </Theme>
    </div>
  }
  dont={
    <div
      style={{
        background: colorsBackgroundDark[3],
        borderRadius: '4px',
        padding: layout.spacingLarge
      }}
    >
      <Theme name="light">
        <SearchInput
          placeholder="Search for people"
          style={{ display: 'block', width: '100%' }}
        />
      </Theme>
    </div>
  }
/>
