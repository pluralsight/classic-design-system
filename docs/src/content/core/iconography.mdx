# Iconography

Create custom icons based on the following guidelines to yield a consistent style. Propose your icons be included in the [common set](/components/icon#common-icon-set) by adding an [issue in Github](https://github.com/pluralsight/design-system/issues/new). For implementation instructions, visit the [Icon component page](/components/icon).

## Design asset

You can find the icon grid template and see all of our icons in the [PSDS - Icons Figma file](https://www.figma.com/file/HVFUT0XlrkfsTMde6PMWLITj/PSDS-Icons).
