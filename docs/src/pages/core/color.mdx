# Color

## Usage

Colors are available from the core package. Colors are usable in CSS:

```css startExpanded
@import "@pluralsight/ps-design-system-core";

.mySelector {
  color: var(--psColorsBlueBase);
}
```

Colors are also usable in JavaScript:

```javascript noRender startExpanded
import {
  colorsBlue,
  /* other named exports */
} from "@pluralsight/ps-design-system-core";

myCssInJsSolution({
  color: colorsBlue.base
})
```
