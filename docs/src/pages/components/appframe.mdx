# App frame

<Intro>
  The App Frame component provides page structure and integrates with other
  navigation components.
</Intro>

## Examples

```typescript noRender
<AppFrame topnav={<TopNav />} sidenav={<CustomSideNav />}>
  {/* page content here */}
</AppFrame>
```

If needed, you can control the the sidenav open state.

```typescript noRender
const [sidenavOpen, setOpen] = useState(false)

const close = () => setOpen(false)
const open = () => setOpen(true)
const toggle = sidenavOpen ? close : open

return (
  <AppFrame
    onRequestSideNavClose={close}
    onRequestSideNavOpen={open}
    topnav={<TopNav onMobileMenuClick={toggle} />}
    sidenav={<CustomSideNav />}
    sidenavOpen={sidenavOpen}
  >
    {/* page content here */}
  </AppFrame>
)
```

## Usage & types

<Usage
  install="npm install @pluralsight/ps-design-system-appframe"
  import="import AppFrame from '@pluralsight/ps-design-system-appframe'"
  packageName="appframe"
/>

<Types.Table>
  <Types.Prop
    name="children"
    type="React.ReactNode"
  />
  <Types.Prop
    name="onRequestSideNavClose"
    type="Event => void"
    desc="callback triggered when the sidenav should be closed. Use if controlling the open state"
  />
  <Types.Prop
    name="onRequestSideNavOpen"
    type="Event => void"
    desc="callback triggered when the sidenav should be opened. Use if controlling the open state"
  />
  <Types.Prop
    name="sidenav"
    type="React.ReactNode | ({ visible }) => React.ReactNode"
    desc="area to display secondary navigation"
  />
  <Types.Prop
    name="sidenavOpen"
    type="boolean"
    desc="control if the sidenav container is opened"
  />
  <Types.Prop
    name="topnav"
    type="React.ReactNode | ({ closeSidenav, openSidenav, sidenavOpen }) => React.ReactNode"
    desc="area to display primary navigation - such as a NavBar"
  />
</Types.Table>
