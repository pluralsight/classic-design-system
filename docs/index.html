Purpose

The Design System strives toward a cohesive design language for Pluralsight’s products, a shared vocabulary for our teams, and basic building blocks to accelerate development.

Core

The Core consists of predefined variables for basic design elements such as color, typography, and spacing. These system properties allow for building layouts and visual styles from defined constraints.

Components

Components include a variety of common UI elements to bootstrap experiences and ensure consistent interaction and style as well as accessibility optimizations . These are React components.
