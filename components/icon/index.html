Examples

Size

Icons can be used at 4 sizes: xSmall (16px), Small (20px), Medium (24px), Large (48px)

import React from 'react'
import { CodeIcon } from '@pluralsight/ps-design-system-icon'


const Comp = () => (
  <div className="example-flex-column">
    <CodeIcon size={CodeIcon.sizes.xSmall} />
    <CodeIcon size={CodeIcon.sizes.small} />
    <CodeIcon size={CodeIcon.sizes.medium} />
    <CodeIcon size={CodeIcon.sizes.large} />
  </div>
)


export default Comp

Color

An Icon will inherit it's color from the font color of it's parent elements in the browser. When using Icon with Text place inside the Text component if you wish for it to inherit color styles. You can override the color by styling the icon with the desired color in css or inline styles as shown below.

import React from 'react'
import { PathIcon } from '@pluralsight/ps-design-system-icon'


const Comp = () => (
  <div className="example-flex-colum">
    <div style={{ color: '#0084BD' }}>
      <PathIcon size={CodeIcon.sizes.large} />
    </div>
    <PathIcon style={{ color: 'rebeccapurple' }} size={CodeIcon.sizes.large} />
  </div>
)


export default Comp
